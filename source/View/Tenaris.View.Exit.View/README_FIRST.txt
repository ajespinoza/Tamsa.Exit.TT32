

Para la ejecucion / depuracion de la aplicacion se requiere que las siguientes librerias
externas al proyecto se encuentren en la carpeta del ejecutable :

Castle.Core.dll
Castle.DynamicProxy2.dll
Microsoft.Practices.Prism.dll
Microsoft.Practices.Prism.Interactivity.dll
Microsoft.Practices.ServiceLocation.dll
PostSharp.dll
PostSharp.Laos.dll
System.Windows.Interactivity.dll
Tenaris.Library.ConnectionMonitor.dll
Tenaris.Library.DbClient.dll
Tenaris.Library.Framework.dll
Tenaris.Library.Log.dll
Tenaris.Library.Proxy.dll
Tenaris.Library.Shared.dll
Tenaris.Library.UI.Framework.dll
Tenaris.Manager.Forum.Shared.dll
Tenaris.Service.Security.Client.dll
Tenaris.Service.Security.dll
Tenaris.Service.Security.LanguageResource.dll
Tenaris.Service.Security.Shared.dll


Para la depuracion solo algunas de ellas son copiadas por el visual studio, las
otras deben ser copiadas de forma manual.
